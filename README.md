## Description

Extended [Nest](https://github.com/nestjs/nest) framework [TypeScript starter repository](https://github.com/nestjs/typescript-starter) - contains *mongoose*, *dotenv* pre-setup with an example *user* module.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
